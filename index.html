Mã hóa/ Giải mã file bằng vigenere

Demo: mã hóa/giải mã file bằng Vigenère (byte-wise) ngay trong trình duyệt. Không gửi file lên server.

Ghi chú

  • Không gửi file ra server — mọi thao tác diễn ra trên máy bạn (trong trình duyệt).
  • Vigenère byte-wise: out[i] = (in[i] ± key[i % keylen]) mod 256.
  • Không dùng cho dữ liệu nhạy cảm. Để an toàn thực sự, dùng AES.
